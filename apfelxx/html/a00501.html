Detailed Description

template<class T>
class apfel::MatchedEvolution< T >

The MatchedEvolution class for the computation of the running of a generic quantity. It can be couplings as well as distributions.

This mother class provides the basic ingredients for the computation and the threshold matching of the running of a given object.
