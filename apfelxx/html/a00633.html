Detailed Description

The NLO matching function classes for FFs.

Reference: arXiv:1604.07869. Notice that the following expressions are implicitly multiplied by a factor x^2. This is because the convolution with FFs have the form:

D(x) = C(x) d(x) / x^2 = (1/x^2) ^1 (dy/y) [y^2 C(y)] d(x/y)
