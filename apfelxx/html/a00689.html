Detailed Description

template<class T>
class apfel::Set< T >

The Set template class.

This class allocates a Set of objects of type T following the ConvolutionMap U. This class provides the methods to deal with operations between different types of objects T.

Things to be improved:

  • there is no real necessity of allocating U
  • missing operators.
