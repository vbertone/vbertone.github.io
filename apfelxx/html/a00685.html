Detailed Description

template<class T>
class apfel::QGrid< T >

Class for the Q-space interpolation QGrids.

Subgrids are the building blocks of the interpolation procedure. This class defines the "QGrid" object that includes, apart from the grid itself, also the relevant parameters.
