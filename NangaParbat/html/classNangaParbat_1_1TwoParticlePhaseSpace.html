Detailed Description

Note
Calculate the phase-space volume within cuts relative to total phase-space for: q -> k + kb with: k_T > kTmin && kb_T > kTmin |eta| < etamax && |etab| < etamax excluding: eta0 < |eta| < eta1 && eta0 < |etab| < eta1

The following class has been extracted from the "Cute" program version 2.0.0 (see https://cute.hepforge.org) and adapted to fit into APFEL++.

Examples
IntegrationTest.cc.
